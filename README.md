# food-website-Rupeshtech2026
a homemade traditional Indian snacks business.

Key Features:
Product Display:

12 authentic South Indian snacks (Bellam Gavvalu, Ariselu, Murukulu, Dry Fruit Laddu, etc.)

Each product shows name, description, price (₹100-250), and "Add to Cart" button

Shopping Cart:

Add/remove items, adjust quantities

Real-time total calculation

Cart count badge on header

WhatsApp Ordering:

"Order Now on WhatsApp" button

Automatically creates order summary message

Opens WhatsApp chat with pre-filled order details

Customer Engagement:

Testimonial slider (auto-rotating)

Floating feedback button with Formspree integration

Contact info & social media links

Visual Design:

Responsive layout (works on mobile/desktop)

Smooth animations, sticky header, back-to-top button

Modern gradient color scheme (warm orange/green theme)

This is a Class 12th/IP project ready for deployment as an online storefront for a small food business.
